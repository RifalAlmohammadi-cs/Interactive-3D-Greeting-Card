🎉 Computer Graphics Project – Interactive 3D Greeting Card
📌 Overview

This project presents an interactive 3D greeting card scene designed to capture the joy of celebration using computer graphics techniques.

At the center of the scene is a decorated table with a three-layer cake, surrounded by gifts, balloons, and party decorations, creating a festive and engaging environment.

🎯 Objectives
Implement 3D shapes and scene composition
Apply texture mapping for realistic surfaces
Use dynamic lighting for visual enhancement
Add interactive features to engage users
Simulate a celebratory environment using graphics
🧠 Features
🎂 3D multi-layer cake
🎁 Gifts and party decorations
🎈 Balloons and festive elements
🎆 Fireworks / animated effects
💡 User-controlled lighting
🎮 Interactive animations
🖥️ Technologies Used
Java
Computer Graphics concepts
3D transformations
Texture mapping
Lighting techniques
